# unityroom投稿用テキスト（下書き）

## 作品タイトル

Link-Shot

## 概要（一覧・カード表示用の短い紹介文）

相手の壁とカードを読み合う、1対1のフリーキック対戦ゲーム。スリングショットでボールを飛ばして得点を競え。

## 説明文（作品ページ本文）

サッカーのフリーキックをモチーフにした、交代制の1対1対戦ゲームです。

5ラウンド制・計10ショットで得点を競います。各ラウンドでは、まず両者が手札から1枚のカードを伏せてセットし、防御側は壁を配置してから、攻撃側がスリングショット操作（引っ張って離す）でボールを発射します。

- カードには3すくみの「属性」があり、攻撃側の属性が防御側に勝つ（または同属性の）場合のみ、カードの効果（壁除去・的の拡大・2連射など）が発動します。相手の手札を読み、噛み合わせを狙ってカードを選びましょう。
- 使い捨て壁カードは全5枚、試合を通して自由なタイミングで使えます。序盤に使い切るか、終盤まで温存するか、駆け引きが生まれます。
- 的は貫通式・ランダム配置。一点集中で高得点を狙うか、まとめて拾って手堅く稼ぐか選べます。

### 対応モード
- **2人対戦**（同一端末で交互に操作）
- **CPU対戦**（弱・強の2段階）
- **オンライン対戦**（ルームコードで友達と対戦、実験的機能）

## 操作方法

- タイトル画面: タップ/クリックで開始
- デッキ選択・カード選択・壁配置: マウスクリック / タップ
- ショット: ボールをドラッグして引っ張り、離して発射（Angry Birds方式）

## 備考

- 横持ち（16:9基準）のPCブラウザ推奨。スマホブラウザにも対応していますが、画面が小さい端末では操作しづらい場合があります
- オンライン対戦は実験的機能です。不具合が見つかった場合はフィードバックいただけると助かります
